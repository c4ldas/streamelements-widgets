<h2 id="widget-name" class="widget-name">Snow widget</h2>

<p id="description" class="description">Add a customizable snowfall effect to your overlay. Control snowflake amount, size, fall speed, wind, and fade behaviour for a dynamic winter atmosphere on stream. Enable or disable it via chat command.</p>
<p>This widget is visually similar to the native StreamElements default widget called <code>Let it snow</code>, but with more options available.</p>

<h2>1-click install:</h2>
<a href="https://streamelements.com/dashboard/overlays/share/695431943bcca07473139b82">https://streamelements.com/dashboard/overlays/share/695431943bcca07473139b82</a></p>

<h2>Preview and instructions</h2>
<h3>How to use it:</h3>
<p>1 - After installing the widget, just play around with the sliders available</p>
<p>2 - You can configure a chat command to enable and disable the widget. The commands are only available for the streamer and moderators to use.</p>
